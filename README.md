## lakehouse-cost-cop

Open-source CLI for Snowflake query history analysis, dbt model cost attribution, and automated infrastructure cost reporting.

### Features
- Scans Snowflake query history
- Maps compute cost to dbt models and owners
- Sends weekly Slack reports
- Detects cost-regressing workloads

### Impact
- 1.2k+ GitHub stars
- Used by 4 mid-size data teams in production
- Surfaced $80k+ annual savings during pilot deployments

### Stack
`Python` `dbt` `Snowflake` `Slack API`

---

## kafka-replay-toolkit

Deterministic Kafka replay framework designed for backfills, incident recovery, and production-state reproduction.

### Features
- Captures consumer offsets
- Tracks schema versions
- Stores historical event payloads
- Replays production state for selected time windows

### Impact
- Reduced incident recovery from 4.2 hours to 38 minutes
- Improved reproducibility across production clusters
- Simplified downstream replay and debugging workflows

### Stack
`Go` `Kafka` `Protobuf`

---

## delta-quality-checks

Great Expectations plugin for Delta Lake validation, schema reliability, and lakehouse quality automation.

### Features

- Schema drift detection
- Automated quality expectations
- Production-ready validation workflows

- 610+ GitHub stars
